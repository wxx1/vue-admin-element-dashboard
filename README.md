# vue-admin-element-dashboard

基于之前做的一个项目，进行简单总结，做的一套前端解决方案

### 下载项目之后，首先安装依赖包
```
npm install
```

### 启动项目
```
npm run serve
```

### 打包项目
```
npm run build
```
